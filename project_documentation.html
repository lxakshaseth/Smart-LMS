1. Teammate Quickstart & Local Environment Setup

1.1 Prerequisites

  • Node.js: Version 18.x or 20.x installed
  • MongoDB: Local MongoDB instance running on mongodb://localhost:27017 OR MongoDB Atlas URI
  • Package Managers: npm or pnpm

1.2 Backend Setup (/backend)

# Step 1: Navigate to backend directory
cd backend

# Step 2: Install dependencies
npm install

# Step 3: Create .env file based on template
cp .env.example .env

# Step 4: Configure environment variables in .env
PORT=5000
MONGO_URI=mongodb://localhost:27017/smart-lms
JWT_SECRET=your_super_secret_jwt_key_here
GROQ_API_KEY=gsk_your_groq_api_key_here
GROQ_MODEL=llama-3.1-8b-instant
GEMINI_API_KEY=your_gemini_api_key_here
YOUTUBE_API_KEY=your_youtube_v3_api_key_here
CLIENT_URL=http://localhost:5173

# Step 5: Start backend server in hot-reload development mode
npm run dev

1.3 Frontend Setup (/frontend)

# Step 1: Open a new terminal and navigate to frontend directory
cd frontend

# Step 2: Install dependencies
pnpm install  # or npm install

# Step 3: Create .env file
cp .env.example .env

# Step 4: Configure VITE environment variables
VITE_API_BASE_URL=http://localhost:5000/api
VITE_SOCKET_URL=http://localhost:5000

# Step 5: Start Vite development server
npm run dev

The application will open on http://localhost:5173.

2. System Architecture & End-to-End Data Flow

2.1 Complete Request-Response Lifecycle

Trace of a Student Action (e.g. Asking AI Tutor)
  1. User Input: Student submits question in AITutor.tsx UI.
  2. Frontend API Client: src/app/lib/api.ts intercepts request, injects Authorization: Bearer from localStorage, and dispatches POST request to /api/ai/solve.
  3. Security Gateway:
    • app.set("trust proxy", 1) enables accurate load-balancer client IP matching.
    • corsOptions validates request origin against allowed domain whitelist.
    • apiLimiter checks rate-limit counter (prevents spam/DDoS).
    • auth.middleware.js decodes JWT, verifies signature, fetches user document, and sets req.user.
  4. Controller Layer: ai.controller.js validates payload, constructs system prompt, and calls ai.service.js.
  5. Service & LLM Engine: safeGroqCall() sends payload to Groq LLaMA 3.1 API with 30s Promise.race() timeout.
  6. Database & State Mutation: Controller calculates XP earned, updates user streak, appends today's weeklyActivity, and updates subject mastery formula in MongoDB.
  7. Real-time Broadcast (Optional): If social/group feature, io.to(room).emit() broadcasts update over Socket.IO.
  8. Client Render: AITutor.tsx parses Markdown via renderMarkdown.tsx and renders solution.

  // Gamification
  xp: { type: Number, default: 0 },

  rank: { type: String, default: "Beginner" },
  streak: { type: Number, default: 0 },
  lastActiveDate: { type: String, default: null },
  
  dailyMission: { type: dailyMissionSchema, default: null },
  weeklyActivity: { type: [weeklyActivitySchema], default: [] },
  subjectStats: { type: [subjectStatsSchema], default: [] }
}, { timestamps: true });

// Pre-save hook: auto-generates username & updates level/rank
userSchema.pre("save", function() {
  this.level = Math.floor(this.xp / 100) + 1;
  if (this.xp >= 2000) this.rank = "Grandmaster";
  else if (this.xp >= 1000) this.rank = "Master";
  else if (this.xp >= 500) this.rank = "Advanced";
  else if (this.xp >= 200) this.rank = "Intermediate";
  else this.rank = "Beginner";
});

8. Mathematical Algorithms Reference Guide

1. Risk Score Engine:
RiskScore = clamp((100 − Accuracy) × 0.4 + (100 − Completion) × 0.3 + StreakImpact × 0.3)
2. Subject Mastery Formula:
Mastery = min(100, Questions × 5 + Notes × 7 + Quizzes × 10)
3. Level & Intelligence Formulas:
Level = ⌊ XP / 100 ⌋ + 1  |  IntelligenceScore = 100 + ⌊ XP × 0.8 ⌋

9. Common Troubleshooting FAQ for Teammates

Q: Why do I get ERR_ERL_UNEXPECTED_X_FORWARDED_FOR error on server start?
This occurs when Express is behind a reverse proxy (like Render or Nginx) and express-rate-limit receives proxy headers without proxy trust enabled. Ensure app.set("trust proxy", 1); remains active in server.js.
Q: Why are CORS preflight OPTIONS failing from local frontend?
Make sure your local frontend URL (e.g. http://localhost:5173) is listed in allowedOrigins inside server.js or added to CLIENT_URL in your .env file.
Q: Tesseract OCR is taking long or failing to initialize?
Ensure eng.traineddata is present in the root/backend folder or accessible via internet so Tesseract.js can load the English OCR language model worker thread.
