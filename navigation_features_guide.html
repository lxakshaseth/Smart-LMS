Smart-LMS Navigation Bar & Interface Feature Manual

Detailed User & Developer Specification for All 14 Sidebar Links, Top Navigation Controls & UI Actions

Complete Interface Blueprint • Version 1.0.0

1. Overview of the Layout Architecture

The Smart-LMS user interface employs a responsive dual-navigation shell consisting of a collapsible Sidebar Navigation Bar on the left and a sticky Top Navbar on the top.

  • Sidebar Component (Sidebar.tsx): Manages primary navigation routing across all 14 core platform modules. Supports collapsible states (256px expanded $\leftrightarrow$ 68px collapsed), animated active indicators, and mobile drawer transitions.
  • Navbar Component (Navbar.tsx): Manages global search queries, theme switching (Light/Dark mode), notification triggers, user profile summaries, target exam selection, and authentication logout flows.

4. Sidebar Controls & Responsiveness Mechanics

Collapsible Sidebar Mechanics (256px ↔ 68px)
Clicking the <ChevronLeft /> or <ChevronRight /> toggle button expands or collapses the sidebar width dynamically using Framer Motion animation. When collapsed, text labels hide smoothly, and floating hover tooltip popovers appear when hovering over icons.
Mobile Hamburger & Overlay Menu
On mobile screens (screen width < 1024px), the sidebar transitions into a slide-out drawer controlled by a fixed top-left hamburger button (<Menu /> / <X />) and a semi-transparent backdrop overlay.
